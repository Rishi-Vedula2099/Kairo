# Kairo ✧

> **Agentic Workflow OS (Zapier + AI Agents)**

Kairo is a high-performance AI orchestration platform that dynamically decides workflow paths, self-heals from failures, and maintains context across complex operations.

## ✧ Vision

Kairo moves away from static, fixed flows. It embraces **Agentic Intelligence**, using a dynamic reasoning engine to capture the "Kairos"—the critical moments of strategic opportunity.

### Core Principles
- **Dynamic Orchestration**: AI decides the next best action in real-time.
- **Neon Ops Dashboard**: A premium, high-visibility workspace for complex logic.
- **Self-Healing Workflows**: Robust retry and fallback logic powered by Temporal.io.

## ✧ Tech Stack

- **Frontend**: Next.js 14, React Flow, Zustand, ShadCN UI.
- **Backend**: Fastify, LangChain, OpenAI.
- **Orchestration**: Temporal.io.
- **Infrastructure**: Docker, PostgreSQL, Redis.

## ✧ Setup

```bash
# Clone the repository
git clone https://github.com/Rishi-Vedula2099/Kairo.git

# Start infrastructure
docker-compose up -d

# Install dependencies
npm install
```
